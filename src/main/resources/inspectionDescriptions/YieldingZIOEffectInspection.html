Detects if a ZIO is returned inside the yield part of a for comprehension. 
for {
    x <- ZIO.succeed(1)
    y <- ZIO.succeed(2)
} yield {
    ZIO.effect(x + y)
}

Will highlight the expression ZIO.effect(x + y).
Fix by moving effect from yield inside of the for: 
for {
    x <- ZIO.succeed(1)
    y <- ZIO.succeed(2)
    res <- ZIO.effect(x + y)
} yield res
